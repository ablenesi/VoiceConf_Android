## Description of the server of the VoiceConf application 
### Table of contents
* Version
* Requirements
* Installation

### Version
**1.0**

### Requirements
 * python 2.7
 * py2exe 0.6.9
 

### Installation
#### Building

``` bash
python setup.py py2exe
```
#### Running

The resulting executable is for command-line use and it does not require any command-line arguments.

``` bash
cd dist
server
```
